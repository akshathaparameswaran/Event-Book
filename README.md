# Event Booking System

A simple Event Booking Website built using HTML, CSS, and JavaScript.

This project allows users to:

- Register an account
- Login using credentials
- View available events
- Book events
- View booked events
- Logout securely

The project uses **LocalStorage** to store user and booking data.

---

# Features

## User Authentication
- User Registration
- User Login
- Logout Functionality

## Event Management
- Display available events
- Book events instantly
- Store bookings in LocalStorage

## Booking Section
- View all booked events
- Auto display saved bookings

## Responsive UI
- Clean and simple design
- Easy navigation

# How the Project Works

## Registration
Users create an account using:
- Name
- Email
- Password

The data is stored in browser LocalStorage.

---

## Login
Users login using:
- Registered Email
- Password

If credentials match, access is granted.

---

## Event Booking
Users can:
- View available events
- Click "Book Now"
- Save bookings in LocalStorage

---

## Logout
Logout removes login session and reloads the page.

---

# LocalStorage Keys Used

| Key | Purpose |
|-----|----------|
| `user` | Stores registered user data |
| `loggedIn` | Stores login session |
| `bookings` | Stores booked events |
